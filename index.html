This is the first visualisation

This map visualization shows the number of death counts per 100000 as of October 31st 2020 for all Asian Countries. As you can see, Bangladesh has the highest Death Counts Per 100000. This is probably caused by the fact the country is economically bad despite being densely populated.

This is the Second visualisation

This ascending ordered bar graph shows the recovery rate of each Asian countries. The recovery rate is calculated by dividing the number of recovered cases by number of positive cases and multiplying it by 100 to get the percentage. From this bar graph it can be concluded that the countries with higher income usually have higher recovery rate because usually they have more access to healthcare services.

This is the third visualisation

This descending ordered bar graph shows the fatality rate of each Asian Countries. The death rate is calculated by dividing the number of death cases by number of positive cases and multiplying it by 100 to get the percentage. From this bar graph it can be seen that Yemen has extremely high fatality rate. This is because Yemen’s already collapsing healthcare system is buckling as the coronavirus pandemic hits an infrastructure already devastated by five years of war. Right now there are most probably only now 10 healthcare workers for every 10 000 people, less than half the World Health Organization benchmark as stated by MedGlobal(2020).
